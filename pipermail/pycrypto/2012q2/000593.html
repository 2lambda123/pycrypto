Both are correct. The fact is that they use different default segment sizes
for CFB:
PyCrypto uses 8 bits, and OpenSSL 64 bits.

With:

bf = Blowfish.new(key = key, mode = Blowfish.MODE_CFB, IV = iv,
       segment_size=64)

You will get the same result.

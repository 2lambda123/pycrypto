Algorithm-specific Notes for Encryption Algorithms

RC5 has a bunch of parameters; see Ronald Rivest's paper at http://theory.lcs.mit.edu/~rivest/rc5rev.ps for the implementation details. The keyword parameters are:

  • version: The version of the RC5 algorithm to use; currently the only legal value is 0x10 for RC5 1.0.
  • wordsize: The word size to use; 16 or 32 are the only legal values. (A larger word size is better, so usually 32 will be used. 16-bit RC5 is probably only of academic interest.)
  • rounds: The number of rounds to apply, the larger the more secure: this can be any value from 0 to 255, so you will have to choose a value balanced between speed and security.
